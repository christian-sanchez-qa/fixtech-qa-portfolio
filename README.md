# Fixtech QA Portfolio

Portfolio de testing QA sobre Fixtech (fixtech.com.ar),
plataforma de e-commerce. Incluye pruebas manuales y de rendimiento
sobre el flujo de compra.

## Proyectos

### 01 - Casos de Prueba
Testing manual del flujo de compra completo.
Smoke, Funcional y Regresión. 20 casos de prueba documentados.

### 02 - Performance Testing con JMeter
Prueba de carga simulando 10 usuarios simultáneos.
Análisis de tiempos de respuesta e identificación de cuellos de botella.
