# Fixtech — Casos de Prueba | Flujo de Compra

## Descripción
Documento de casos de prueba sobre el flujo de compra de Fixtech
(fixtech.com.ar), cubriendo los principales escenarios funcionales
y no funcionales.

## Herramienta
- Microsoft Excel

## Tipos de Testing
- Smoke Testing
- Testing Funcional
- Testing de Regresión

## Flujo Testeado
1. Selección de producto
2. Carrito de compras
3. Verificación de email
4. Ingreso de código postal
5. Pago y confirmación
